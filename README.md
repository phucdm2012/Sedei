#  Sedei

If you are Windows user and you dont know what tool that can help you do your work more faster with your computer, this application can help you.

when I was thinking about macOS, I had an idea: make an application to manage files / folders by tags and color of tags. I think it can help you do your work faster because you can mark your files and folers byt theme. For example, you can use red tag to mark files about work, green tag for films, yellow tag for games, blue tag for programming, pink tag for picture, etc.

Beside, you can make more tag with your custom tag name and color.
